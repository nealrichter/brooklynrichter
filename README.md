# brooklynrichter

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/nealrichter/brooklynrichter.git
cd brooklynrichter
```

### Create a New Repository from Scratch

If you're setting up this repo for the first time:

```bash
# Create a new directory and initialize git
mkdir brooklynrichter
cd brooklynrichter
git init

# Create a README
echo "# brooklynrichter" > README.md

# Stage and commit
git add README.md
git commit -m "Initial commit"

# Link to GitHub and push
git remote add origin https://github.com/nealrichter/brooklynrichter.git
git branch -M main
git push -u origin main
```
